JavaScript audio and timed lyrics synchronizer.

== Description ==

Rabbit Lyrics features:

- Three different view modes: clip, mini, full
- Music and audio book supports
- Translation supports
- Karaoke highlighting
- Jump to playback position by clicking lines
- Built-in themes that fits official WordPress themes
